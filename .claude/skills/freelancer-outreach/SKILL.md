# Freelancer Outreach

Run this when Howard wants to find leads or draft pitches for freelance work.

## Steps

1. **Find leads:** Research platforms and opportunities:
   - Upwork, Fiverr, Freelancer (web dev gigs)
   - Facebook groups, Reddit, Discord servers where people need websites
   - Local businesses in Cavite that might need a web presence
   - Search for "need a website" or "looking for web developer" posts

2. **Qualify leads:** For each lead, note:
   - What they need
   - Estimated budget range
   - How urgent it seems
   - Whether it matches Howard's current skills (HTML/CSS + Claude Code)

3. **Draft pitch:** Write a short, confident pitch tailored to the lead. Template:
   ```
   Hey [name],

   I saw you're looking for [what they need]. I build clean, fast websites
   and can have [deliverable] ready by [timeline].

   Here's what I'm thinking: [1-2 sentences on approach].

   Want to hop on a quick call to talk details?

   -- Howard
   ```

4. **Review:** Show Howard the leads and pitches for approval before sending.

## Output Format

```
**Leads found:** [number]
**Top picks:**
1. [lead name/source] -- [what they need] -- [est. budget]
2. [lead name/source] -- [what they need] -- [est. budget]

**Draft pitch for #1:**
[pitch text]
```
