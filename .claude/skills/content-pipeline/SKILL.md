# Content Pipeline

Run this when Howard wants to create content (YouTube video, social post, etc.).

## Steps

1. **Idea:** Ask Howard what the topic or idea is. If he doesn't have one, brainstorm 3-5 ideas based on:
   - His niche (automation, web dev, freelancing journey)
   - Trending topics in tech/content creation
   - His current projects (clip automation, web dev freelancing)

2. **Script/Outline:** Create a structured outline:
   - Hook (first 5 seconds -- why should someone care?)
   - Main points (3-5 key things to cover)
   - Call to action (subscribe, follow, check out a link)

3. **Draft:** Write the full script or post copy. Match Howard's voice: concise, confident, sounds like his own words.

4. **Review:** Show Howard the draft. Revise based on feedback.

5. **Schedule:** Help Howard plan when to publish. Keep a simple content calendar if multiple pieces are in the pipeline.

## Output Format

```
**Idea:** [one-liner]
**Format:** [YouTube video / short / post]
**Outline:**
- Hook: [...]
- Point 1: [...]
- Point 2: [...]
- Point 3: [...]
- CTA: [...]
**Draft:** [full script or copy below]
```
