# Daily Task Reminder

Run this when Howard needs a reminder of his daily personal habits.

## Daily Habits Checklist

- [ ] Make your bed
- [ ] Walk Chopper
- [ ] Work out
- [ ] Eat something healthy

## How to Use

When Howard asks for his daily tasks or when running the Morning Kickoff skill, include this checklist. Keep it simple -- just the list, no extra commentary unless he asks.

## Output Format

```
**Daily tasks:**
- [ ] Make your bed
- [ ] Walk Chopper
- [ ] Work out
- [ ] Eat something healthy
```
