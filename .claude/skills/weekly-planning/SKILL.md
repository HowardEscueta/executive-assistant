# Weekly Planning

Run this at the start of each week (or when Howard says "let's plan the week").

## Steps

1. **Review last week:**
   - What got done?
   - What didn't get done? Why?
   - Any decisions made? (Check `decisions/log.md`)

2. **Check the bigger picture:**
   - Read `context/goals.md` -- where are we on Q2/Q3 milestones?
   - Read `context/current-priorities.md` -- still accurate?
   - Check active projects in `projects/` -- any status changes needed?

3. **6-month progress check:**
   - Howard's 6-month deadline is ~September 2026.
   - Are we on track? Behind? Ahead?
   - What needs to shift?

4. **Set this week's goals:**
   - Pick 3-5 specific, actionable goals for the week.
   - Each goal should tie back to the quarterly milestones or active projects.

5. **Update files if needed:**
   - Update `context/current-priorities.md` if focus has shifted.
   - Update project READMEs if status changed.

## Output Format

```
**Last week recap:** [quick bullets]
**6-month check:** [on track / behind / ahead + why]
**This week's goals:**
1. [goal]
2. [goal]
3. [goal]
**Anything to update?** [file changes needed]
```
